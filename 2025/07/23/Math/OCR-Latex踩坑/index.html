【心碎踩坑王】OCR-Latex踩坑

在 mac 上尝试安装 Latex-OCR，哪知道 LaTeX-OCR 拉了坨大的（官方方法根本就不支持 macOS），恶心坏了。按照官方文档方法安装一个小时多，又是 pip 又是 docker 的，一会儿 pip 3，一会儿 pip 的。整了半天一直卡在最后一步 run 无法运行。

无奈放弃官方文档的方法 – “m 1 安装 latex-OCR…”，一搜一个准，很快啊。找到民间的教程：https://blog.rongxin.me/install-latex-ocr-on-apple-silicon-m1

好家伙，一次成功。

具体步骤

  1. pip install "pix2tex[gui]" 安装模块
  2. brew install pyqt@5 安装所需要的 GUI 模块
  3. sudo cp -r /opt/homebrew/Cellar/pyqt@5/5.15.7_2/lib/python3.9/site-packages/* /Users/rey/miniconda3/lib/python3.9/site-packages/ 因为你在pip或conda的官方论坛上找不到兼容版本。
  4. pip install pynput screeninfo 包依赖
  5. conda install pytorch torchvision conda 提供的兼容 pytorch

使用方法

  1. 启动 python -m pix2tex latexocr
  2. 截图需要识别的 LaTeX 图片至剪贴板
  3. 在终端内按下 ENTER，待有反应时 latex 已存入剪切板。

对原作者的补充

若是使用 zsh 终端，可按照该方法提高开启 OCR 效率。

添加代码片：

:wq 保存并退出，并编译 source ~/.zshrc。以后便可直接在终端中 ocr 来直接开启工具，方便快捷，独家技巧。
